Cara Top Up di LDM

Panduan lengkap top up game di LuksDiaMon — mudah, cepat, aman!

1
Pilih Game & Nominal
Pilih game yang ingin kamu top up dari halaman beranda, lalu pilih jumlah nominal yang kamu inginkan.
💡 Cek kolom TERLARIS dan HEMAT untuk nilai terbaik!
2
Masukkan ID Game
Masukkan User ID dan Zone ID (jika diperlukan) akun game kamu dengan benar. Jangan sampai salah!
💡 Cara cek ID:
Mobile Legends: Buka game → tap profil → lihat angka di bawah nama
Free Fire: Buka game → tap profil di pojok kiri atas
PUBG Mobile: Buka game → tap profil → lihat ID di bawah nama
3
Pilih Metode Bayar
Pilih metode pembayaran yang kamu mau: GoPay, DANA, OVO, transfer bank, atau QRIS.
💡 QRIS bisa dipakai dari semua e-wallet manapun!
4
Lakukan Pembayaran
Transfer sesuai nominal yang tertera. Pastikan jumlahnya tepat agar sistem bisa memverifikasi otomatis.
⚠️ Jangan lebih, jangan kurang dari nominal yang tertera!
5
Diamond Masuk!
Setelah pembayaran terverifikasi, diamond/item akan langsung masuk ke akun game kamu dalam hitungan detik.
✅ Simpan nomor order kamu untuk pengecekan status transaksi.

FAQ

Berapa lama proses top up?
Proses top up berlangsung dalam 1–5 menit setelah pembayaran terverifikasi. Pada jam sibuk bisa sampai 15 menit.
Bagaimana kalau ID game saya salah?
Jika ID game salah dan top up sudah diproses, kami tidak bisa membatalkan transaksi. Pastikan ID sudah benar sebelum bayar!
Apakah pembayaran aman?
Ya, semua transaksi diproses dengan aman. Data kamu tidak akan kami bagikan ke pihak manapun.
Bagaimana kalau diamond tidak masuk?
Hubungi CS kami via WhatsApp dengan menyertakan nomor order dan bukti pembayaran. Kami akan segera menangani dalam 1x24 jam.
Masih ada pertanyaan?
Hubungi CS kami langsung via WhatsApp, siap membantu 24/7!

Kini, 4 tahun kemudian, LuksDiaMon telah melayani lebih dari 2 juta transaksi dari seluruh Indonesia. Kepercayaan kalian adalah energi kami untuk terus berkembang dan memberikan layanan terbaik.
